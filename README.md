# goit-algo-hw-02

## Домашнє завдання до теми 2. “Основні структури даних”, Завдання 1

Програма в [modul1.py](modul1.py) імітує роботу сервісного центру:

- автоматично генерує нові заявки;
- додає їх у чергу (Queue з модуля queue);
- послідовно обробляє заявки за принципом FIFO;
- виводить повідомлення, якщо черга порожня.

## Відповідність псевдокоду

У рішенні реалізовано:

- створення черги заявок;
- функцію generate_request();
- функцію process_request();
- головний цикл, який постійно викликає генерацію та обробку заявок.

Кожна заявка має:

- унікальний ідентифікатор id;
- умовне ім'я клієнта.

## Запуск

1. Перейдіть у папку проєкту.
2. Запустіть програму командою:

```bash
python modul1.py
```

1. Для завершення роботи натисніть Ctrl+C.

## Очікувана поведінка

Під час виконання програма показує:

- створення нових заявок;
- обробку заявок у порядку їх надходження;
- повідомлення Queue is empty, якщо немає заявок для обробки.

## Завдання 2 — Перевірка паліндрома

Програма в [modul2.py](modul2.py) визначає, чи є переданий рядок паліндромом, використовуючи двосторонню чергу `deque` з модуля `collections`.

### Реалізація

- Рядок нормалізується: переводиться до нижнього регістру, пробіли видаляються.
- Усі символи додаються до `deque`.
- Символи порівнюються з обох кінців черги до тих пір, поки не залишиться один або жоден символ.
- Підтримуються рядки з парною та непарною кількістю символів.

### Як запустити

```bash
python modul2.py
```

### Приклади виводу

```text
"racecar" -> паліндром
"A man a plan a canal Panama" -> паліндром
"hello" -> не паліндром
"Never odd or even" -> паліндром
"abba" -> паліндром
```
